# Joycon charge PCB



This folder contains the KiCad files for the optional front buttons for system power, monitor

power, volume up, and volume down.



Have these boards made by a PCB manufacturer of your choice.  Solder on a 2 conductor wire

as well as a FH36W-11S-0.3SHW(99), available here:

[https://www.digikey.com/en/products/detail/hirose-electric-co-ltd/FH36W-11S-0-3SHW-99/4168641](https://www.digikey.com/en/products/detail/hirose-electric-co-ltd/FH36W-11S-0-3SHW-99/4168641)
