# D-Pad: A Handheld Framework Gaming PC
Start with a 3D printed case, Add some laptop parts from Framework Computer Inc [https://frame.work/](https://frame.work/) as well as some other electronics and hardware and what do you have?  A fully customizabe, fully upgradeable, and fully repairable handheld gaming PC that you can make your own!

## Features
Able to use any Framework 13 Motherboard and Battery
8.9" IPS Touchscreen
Front facing speakers
Joycon docking/charging
Framework Expansion Card compatibility
Docking station connector
Power and volume buttons
Kickstand
Additional expansion provisions

## Building Your Own
The build process is outlined in a YouTube video (link to video)
I recommend watching it completely at least once to get an idea of how D-Pad is assembled.

You should be able to find all the other information you need to build one in this repository.
